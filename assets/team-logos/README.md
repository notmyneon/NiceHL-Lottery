# NiceHL Team Logo Library

This folder contains the full team-logo library.

Recommended rule:
- Keep one folder per team.
- Upload each real team logo as `logo.png`.
- The page now defaults to `logo.png`.
- If `logo.png` is missing, it will try the placeholder `logo.svg` as a backup.

Example:
```text
assets/team-logos/rockaway-brattwursts/logo.png
assets/team-logos/steel-valley-fishercats/logo.png
```

The page includes a `TEAM_LOGO_LIBRARY` in `index.html`.
That library connects exact team names and aliases to these logo paths.
